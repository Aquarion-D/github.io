About Aquarion

Aquarion is a customizable Discord theme. Clean. Stylish. No telemetry. Just vibes.

Download

You can download the latest version from our GitHub.

Contributors

  • NyxIsBad — Helped get this up and running!

Privacy Notice

Aquarion does not collect any user data or telemetry.

When checking for updates, Aquarion will access:

  • https://aquarion-d.github.io/Aquarion/main.css
  • https://discordstyles.github.io/RadialStatus/RadialStatus.css

GitHub may show public stats like downloads and forks.

Changelog for Aquarion v1.5.5

  • UI Enhancements:
    • Moved date dividers to the middle.
    • Added new alt-color options for customization.
    • Cleaned up Nitro "most popular" section (removed border).
    • Updated activity texts to use alt-color.
    • Reworked message pill (centered, square, alt-color).
    • Redesigned server member list with alt-color roles and main-color borders.
    • Server category names now use alt-color.
    • Reduced borders and added shading for a cleaner look.
    • Changed wordmark to “Aquarion v1.5.5” (removed glow).
    • Updated theme button whites to match main color.
  • Vencord Plugin Updates:
    • Adjusted the Spotify controls border.
    • Darkened plugin boxes and changed info text to white.
    • Added support for voice messages in Vencord.
    • Made background for the home icon transparent for cleaner visuals.
    • Removed the home shop mosaic.
    • Darkened voice count slightly for better readability.
  • Interaction Improvements:
    • Tweaked the Nitro button shine (slowed down and applied alt-color).
    • Refined the message request section style for better visibility.
    • Adjusted keybind recording for voice/video push-to-talk for clearer display.
  • Miscellaneous Changes:
    • Made the "Looking for blocked users?" pop-up transparent.
    • Reduced the flashiness of forums list title and user/message content.
    • Added hover effect to emoji removal in server settings.
    • Modified the boost popup and server boost settings for clarity.
    • Applied background overlay to boost mural in server menu.
    • Added a frosted glass effect toggle in settings.
  • Other Improvements:
    • A bunch of small tweaks for better aesthetics and usability, details can be found in the commits.
